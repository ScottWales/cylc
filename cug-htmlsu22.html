7.5 Plotting The QuickStart.a Dependency Graph

Right-click on the QuickStart.a suite in gcylc and choose Graph; or by command line,

% cylc graph QuickStart.a 2011052300 2011052318 &

This will pop up a zoomable, pannable, graph viewer showing the graph of Figure 20. If you edit the suite.rc file the viewer will update in real time whenever you save the file.


Figure 20: The QuickStart.a dependency graph, plotted by cylc.
