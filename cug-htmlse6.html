6 Suite Registration

Before you can do anything to a suite with cylc commands you must register it in a suite database. Cylc commands target particular suites via their registered names so that you don’t need to remember and continually re-type the actual location of the suite definition directory on disk. Suite names are hierarchical like directory paths but delimited by ‘.’ (foo.bar.baz), allowing suites to be grouped and organised into trees (c.f. directory trees). Task groups are virtual and do not need to be explicitly created before use or removed if they contain no tasks.
