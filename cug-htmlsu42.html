10.1 Advanced Topics

  • spin-up processes via temporary tasks and adding prerequisites on-the-fly
  • how to recover from certain kinds of task failure
  • instant simulation mode clones of a running suite
  • fuzzy prerequisites - triggering of the most recent available, within limits, instance of an upstream task
  • asynchronous tasks (no cycle time) - for running a tree of tasks initiated by a random event such as a satellite pass.
  • subsuites - running another suite inside a task:
    [tasks] 
        [[foo]] 
            command = "cylc␣run␣SUITE␣$CYCLE_TIME␣--until=$CYCLE_TIME"
