3.1 Known Version Compatibility Issues

Cylc should run out of the box on reasonably recent Linux distributions.

For distributed suites the Pyro versions installed on all suite or task hosts must be mutually compatible. Using identical Pyro versions guarantees compatibility but is generally not necessary.

Recent versions of Pyro require Python 2.5 or greater, due to use of the ‘with’ statement introduced in 2.5.

Beware of old Linux distributions that come packaged with Pyro 3.9 or earlier, which is not compatible with the new-style Python classes used in cylc. It has been reported that Ubuntu 10.04 (Lucid Lynx), released in September 2009, suffers from this problem.
